Capabilities

Everything you need. Nothing you don't.

Six pillars of server management, unified under a glass-dark interface with AI woven into every surface.

>_

Terminal

Full PTY terminal in the browser. xterm.js with 256-color support, resize, and session persistence. Run any command without leaving the dashboard.

◓

AI-Powered DB Studio

Supabase-style database management. SQL editor with autocomplete, table browser, schema explorer, migration manager, and Claude-powered query generation.

▦

Docker Fleet

Native Docker Engine API integration. Start, stop, inspect, and prune containers. Real-time stats, log streaming, and AI fleet analysis.

⇨

Git + Deploy

Rich git intelligence center with commit heatmaps, branch management, and AI code review. One-click deploy pipeline with pre-flight checks and rollback.

☉

Security Center

Posture scoring, secret scanning, dependency audit, and encrypted credential vault. AES-256-GCM encryption at rest. AI-generated fix recommendations.

◇

Monitoring

Real-time CPU, memory, and disk metrics. Uptime checks with configurable intervals. Multi-server aggregation. Cron job management with natural language scheduling.
