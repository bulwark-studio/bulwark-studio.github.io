Capabilities

Everything you need. Nothing you don't.

Six pillars of server management, unified under a glass-dark interface with AI woven into every surface.

>_

Terminal

Full PTY terminal in the browser. xterm.js with 256-color support, resize, and session persistence. Run any command without leaving the dashboard.

◓

AI-Powered DB Studio

Supabase-style database management. SQL editor with autocomplete, table browser, schema explorer, migration manager, and Claude-powered query generation.

▦

Docker Fleet

Native Docker Engine API integration. Start, stop, inspect, and prune containers. Real-time stats, log streaming, and AI fleet analysis.

⇨

Git + Deploy

Rich git intelligence center with commit heatmaps, branch management, and AI code review. One-click deploy pipeline with pre-flight checks and rollback.

☉

Security Center

Posture scoring, secret scanning, dependency audit, and encrypted credential vault. AES-256-GCM encryption at rest. AI-generated fix recommendations.

◇

Monitoring

Real-time CPU, memory, and disk metrics. Uptime checks with configurable intervals. Multi-server aggregation. Cron job management with natural language scheduling.

Overview

AI-Powered Dashboard

98/100 health score, AI briefing, command hub with one-click diagnostics, deploy checks, and live CPU/memory/disk metrics at a glance.

DevOps

Terminal AI

Full PTY terminal with AI assistant. Natural language commands — "SSH into production," "check disk space," "restart Docker containers." Shell, AI, and Vault tabs.

Database

SQL Editor

Supabase-style query editor with syntax highlighting, autocomplete, query history, saved queries, Ask Claude AI generation, and CSV export.

Infrastructure

Docker Fleet

Native Docker Engine API. Container stats, log streaming, multi-host connections, AI fleet intelligence. Start, stop, restart, inspect from the dashboard.

Database

Table Browser

Two-panel explorer with columns, data, constraints, foreign keys, and indexes tabs. Filter tables, inspect schemas, open in SQL Editor.

Operations

Kanban Tickets

Drag-and-drop ticket board with AI triage. Pending, Analyzing, Fixing, Testing, Awaiting Approval, Approved, Deployed — full lifecycle tracking.

Security

Roles & Permissions

PostgreSQL role manager with AI security audit, permission heatmap, table-level CRUD matrix. Create least-privilege roles with AI generation.

Monitoring

Real-Time Metrics

AI performance analysis, CPU/memory/connections/uptime gauges, process telemetry table with PID, user, CPU%, memory, and command details.
